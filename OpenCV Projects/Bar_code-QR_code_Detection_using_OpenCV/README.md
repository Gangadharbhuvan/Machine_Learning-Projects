# Bar_code-QR_code_Detection_using_OpenCV

To run this code,
first install some packages
```
pip install opencv-python
pip install pyxbar
```

Then run
```
python Bar_QR-Code_Detection.py
```

Detect the input images or try with Webcam to detect the QR and Bar code
